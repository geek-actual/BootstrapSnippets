Bootstrap Snippets for Visual Studio

Free, open source HTML snippets for Twitter Bootstrap that you can use within Visual Studio.

Simply type the snippet in Visual Studio and press [tab] to expand

btemplate: Basic Bootstrapped template including the appropriate CSS and JS files

http://twitter.github.com/bootstrap/getting-started.html#html-template

<!DOCTYPE html>
<html>
<head>
    <title>Bootstrap 101 Template</title>
    <!-- Bootstrap -->
    <link href="css/bootstrap.min.css" rel="stylesheet" media="screen">
</head>
<body>
    <h1>Hello, world!</h1>
    <script src="http://code.jquery.com/jquery-latest.js"></script>
    <script src="js/bootstrap.min.js"></script>
</body>
</html>

Setup

It's easy to setup BootstrapSnippets into your Visual Studio environment

  1. Download the latest release of BootstrapSnippets
  2. Unzip and open the snippets folder
  3. Copy all the snippets into %MyDocuments%\Visual Studio 2012\Code Snippets\Visual Web Developer\My HTML Snippets
  4. Restart Visual Studio

Note: Replace \Visual Studio 2012\ with any version(s) of Visual Studio that should use the snippets
